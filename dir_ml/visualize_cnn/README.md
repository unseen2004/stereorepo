# Visualize CNN

Visualization tools for Convolutional Neural Networks with sample data and trained models.

---

## Overview

Interactive visualization of CNN architectures, activations, and filters. Helps understand how convolutional neural networks learn and process image data.

---

## Requirements

Install dependencies:
```bash
pip install -r requirements.txt
```

---

## Running

Launch the Jupyter notebook:
```bash
jupyter notebook main.ipynb
```

---

## Features

- CNN architecture visualization
- Layer activation visualization
- Filter/kernel visualization
- Feature map inspection
- Interactive exploration of network internals

---

## Contents

- `main.ipynb` - Main visualization notebook
- `requirements.txt` - Python package dependencies
- Sample models and datasets
