# Digits Recognizer

Handwritten digit recognition using machine learning techniques.

---

## Overview

Machine learning project for recognizing handwritten digits from image data. Implements various classification algorithms to identify digits 0-9.

---

## Requirements

Install Jupyter and required packages:
```bash
pip install jupyter numpy pandas scikit-learn matplotlib
```

---

## Running

Launch the Jupyter notebook:
```bash
jupyter notebook digitsRecognizer.ipynb
```

---

## Features

- Digit classification from pixel data
- Multiple ML algorithms comparison
- Visualization of results and predictions
- Model evaluation and accuracy metrics
