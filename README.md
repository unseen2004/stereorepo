# Learning

A multi-repository collection of learning projects covering various technologies and domains.          
Use https://github.com/unseen2004/mono-clone to clone specific project.

---

## Structure

### [cocktails-api](./cocktails-api)
TypeScript/Node.js REST API for cocktail management with Prisma ORM, authentication, and testing suite.

### [dir_cybersecurity](./dir_cybersecurity)
Cybersecurity projects including cryptography tools and file integrity monitoring.

### [dir_ml](./dir_ml)
Machine learning and AI projects featuring clustering algorithms, neural networks, and game bots.

### [dir_rust](./dir_rust)
Rust learning projects including CLI tools, pattern matching utilities, and queue systems.

### [ecommerce_project](./ecommerce_project)
Full-stack e-commerce application with Node.js backend and React frontend.

### [py_jp](./py_jp)
Python framework for Japanese language programming with custom syntax and preprocessor.

### [slice_orch](./slice_orch)
Rust-based 5G network slice orchestrator with gRPC services and telemetry.

### [webserver_cache_rdma](./webserver_cache_rdma)
High-performance C++ web server with LRU cache and optional RDMA support.
