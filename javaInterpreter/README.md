# jlox

A Java implementation of the Lox interpreter from  Robert Nystrom - "Crafting Interpreters".

## Build and Run

### Windows
1. Run `build.bat` to compile.
2. Run `run.bat <script_file>` to execute a script (or just `run.bat` for REPL).

### Linux / macOS
1. Make scripts executable: `chmod +x build.sh run.sh`
2. Run `./build.sh` to compile.
3. Run `./run.sh <script_file>` to execute a script (or just `./run.sh` for REPL).
