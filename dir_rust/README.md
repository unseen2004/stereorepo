# Rust Projects

Collection of Rust learning projects including CLI tools, pattern matching utilities, and systems programming.

---

## Projects

### [minigrep](./minigrep)
Simple grep-like text search tool implementation in Rust.

### [pattern_search](./pattern_search)
Fast, idiomatic Rust grep-like CLI with streaming IO and rich feature flags.

### [queue_system](./queue_system)
Queue management system with QR code generation and OpenCV integration.

### [rust_cli](./rust_cli)
Command-line interface utilities and tools in Rust.

---

## Overview

These projects demonstrate Rust programming fundamentals including:
- Command-line argument parsing
- File I/O and text processing
- Error handling with Result types
- Systems programming concepts
- External crate integration

---

## Building

Each project can be built with Cargo:
```bash
cd <project_directory>
cargo build --release
```

---

## Running

Execute with Cargo:
```bash
cargo run -- [arguments]
```

Or run the compiled binary directly from `target/release/`
